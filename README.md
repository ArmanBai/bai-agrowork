# BAi AgroWork

**BAi AgroWork** is a prototype platform for agricultural workforce matching in Kazakhstan.

The product is focused on connecting employers in the agricultural sector with seasonal workers, machine operators and rural workforce.

## Problem

Agricultural employers often search for seasonal workers manually through chats, personal contacts and informal channels. This creates problems with trust, quality, availability, ratings, payment discipline and repeated hiring.

## Product concept

BAi AgroWork is designed to help:

- employers create job requests;
- workers find suitable seasonal work;
- both sides build rating and trust history;
- employers invite workers;
- workers accept or decline jobs;
- completed work affects rating and reliability.

## Current status

JavaFX/Maven desktop MVP prototype is under development.

## Planned features

- Worker profile
- Employer profile
- Job requests
- Matching logic
- Invitations and responses
- Work statuses
- Payment status
- Ratings
- Complaints
- Trust indicators
